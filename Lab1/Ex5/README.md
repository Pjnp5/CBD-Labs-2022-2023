**Estruturas**
<br>
* Class Main, menu/interface do sistema de mensagens
* class users_set, esta classe é usada para manipular a base de dados responsável pelas subscrições, usando Sets em Redis
* class user_List, esta classe é usada para manipular a base de dados responsável pelas mensagens, usando listas em Redis
* class user_Hash, esta classe é usada para manipular a base de dados responsável pelos informções dos utilizadores, usando hash em Redis
